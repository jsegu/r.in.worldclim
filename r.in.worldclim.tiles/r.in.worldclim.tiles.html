DESCRIPTION

r.in.worldclim.tiles finds the tiles from WorldClim (current climate) that cover an user defined region of interest (ROI) and optionally downloads them.

WorldClim (version 1.4) provides data layers of the average monthly climate data for minimum, mean, and maximum temperature and for precipitation for 1960-1990. In addition, one can download derived bioclimatic variables.

This addon is to download tiles for a specific region that can be defined using a vector or raster layer or a saved region. After downloading, one can use the addon r.in.worldclim to import the layers in your GRASS GIS database.

If you use this addon, you are strongly encouraged to visit the Worldclim website, for more information about the methods used to generate the climate layers, the units and formats of the data and the names of developers.

If you download and use the data, make sure to cite the data:

Hijmans, R.J., S.E. Cameron, J.L. Parra, P.G. Jones and A. Jarvis, 2005. Very high resolution interpolated climate surfaces for global land areas. International Journal of Climatology 25: 1965-1978.

EXAMPLES

Download the tiles for monthly rainfall for Nepal 
r.in.worldclim.tiles vector=Nepal variables=prec

Determine the tiles for temperature and precipitation data for central Europe, using a raster layer to define the region of interest. As output the addon provides a list of the tiles that you can download. 

r.in.worldclim.riles variables=tmean,prec raster=centr_europe
